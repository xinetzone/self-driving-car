# 无人驾驶简介

参考：{github}`daohu527/awesome-self-driving-car`

## 自主驾驶开源技术

### 论文

- [End to End Learning for Self-Driving Cars](https://arxiv.org/abs/1604.07316)

### 全栈

- [apollo](http://apollo.auto/)：Apollo 是一个开源的自动驾驶平台，包括硬件、系统、车辆平台、云服务等。你可以通过 Apollo 快速建立一个属于你自己的自动驾驶系统。
- [Autoware.Auto](https://www.autoware.org/autoware-auto)：Autoware.Auto 以 ROS 2 为基础，由一个开源社区经理管理，应用一流的软件工程实践。

### 库

- [ROS2](https://docs.ros.org/en/galactic/index.html)：机器人操作系统（ROS2）是一个用于编写机器人软件的灵活框架。它是一个工具、库和惯例的集合，旨在简化在各种机器人平台上创建复杂和强大的机器人行为的任务。
- [OpenCV](https://opencv.org/)：OpenCV（Open Source Computer Vision Library，开源计算机视觉库）是一个开源的计算机视觉和机器学习软件库。OpenCV 的建立是为了给计算机视觉应用提供一个通用的基础设施，并加速机器感知在商业产品中的应用。
- [点云库](http://pointclouds.org/)：点云库（Point Cloud Library，PCL）是一个独立的大型开放项目，用于 2D/3D 图像和点云处理。广泛用于处理激光点云数据。
- [ompl](https://ompl.kavrakilab.org/)：开放式运动规划库。
